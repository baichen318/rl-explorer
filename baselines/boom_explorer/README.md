# boom-explorer
BOOM Explorer: BOOM Microarchitecture Design Space Exploration

## End-to-end Flow
- Get help
```bash
$ cd /path/to/baselines/boom_explorer
$ python main.py -h
```
- Run BOOM-Explorer
```bash
$ cd /path/to/baselines/boom_explorer
$ python main.py -c ../../main/configs/example.yml -s configs/boom-explorer.yml
```
